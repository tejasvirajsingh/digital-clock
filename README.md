# ⏰✨  Digital Clock  ✨⏰

A **beautiful** and **lightweight** digital clock that shows the current time in **12-hour format** with AM/PM — perfect for your website or personal projects! 🕒💻

---

## 🌟 Features
✅ Real-time clock updates **every second**  
✅ 12-hour format with clear **AM/PM** indicator  
✅ **Zero-padded** hours, minutes, and seconds  
✅ Simple, minimal, and **easy to integrate**  

---

🚀 (https://digitalclockmax.netlify.app) 🚀

## 💻 How to Run Locally
1. 📥 **Clone** this repository.
2. 📂 Open the `index.html` file in your browser.
3. 🎉 Enjoy your **live** 12-hour format clock.
